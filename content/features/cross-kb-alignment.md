# Cross-KB Alignment

*Concept*

Align entities across knowledge bases. Entity matching via exact IRI match and embedding similarity. IRI remapping for entities that represent the same concept across KBs. Align tab in review page. KbPicker selects source KB, AlignmentCard shows each match with accept/reject.
