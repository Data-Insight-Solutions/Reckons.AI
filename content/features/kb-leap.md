# KB Leap

*Concept*

Cross-reference entities between KBs. A leap node stores a target (KB stable ID, app path, or URL) as an ordinary RDF triple. Nodes with leaps show an amber ring. Click to jump to the target KB, navigate within the app, or open an external URL. Docs sub-graphs auto-import on first click.
