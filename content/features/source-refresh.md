# Source Refresh

*Concept*

Generic refresh for url, repository, and calendar sources. Auto-refresh on open and on interval (configurable). KB page refresh button. MCP tools: kb_list_sources and kb_request_refresh. Delta comparison shows what changed since last ingest.
