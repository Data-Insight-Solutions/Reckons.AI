# Advantage: MCP Queryability

*Concept*

kb_search finds relevant entities across all graphs with a single query. No need to know which file to read. kb_compress reduces tokens by 60-70%. kb_reckoning gives grounded STP analysis. This is fundamentally better than reading 15 markdown files.
