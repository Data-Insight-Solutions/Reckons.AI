# MCP Server

*Concept*

Standalone Node.js MCP server exposing your KB to AI agents (Claude Desktop, Cursor). 6 tools: kb_search (BM25), kb_get_entity, kb_list_entities, kb_stats, kb_add_note, kb_reckoning. Reads knowledge.ttl from workspace folder.
