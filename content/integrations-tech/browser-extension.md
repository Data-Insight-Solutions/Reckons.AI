# Browser Extension

*Concept*

Manifest V3 extension with three tabs: Compare (page vs KB with at-a-glance bar), Session (multi-page research accumulation with aggregate summaries), Ingest (batch send to app). Supports Chrome, Firefox desktop, and Firefox for Android with mobile-optimized UI.
